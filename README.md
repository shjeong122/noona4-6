

# HTML&CSS 반응형 웹 사이트

## 🚀 프로젝트 개요
소개: 다양한 프로젝트를 팀 매칭 플랫폼</br>
목표: 다양한 해상도(모바일/태블릿/데스크탑)에 대응하는 정적 웹사이트 제작</br>
핵심 포인트: 시맨틱 마크업, 컴포넌트화된 CSS, Bootstrap 유틸리티를 활용한 빠른 레이아웃 구성

## 👥 팀 구성
팀원: 4명
담당 역할: Scrum Master

## 🧭 담당 업무
- **프로젝트 관리 및 협업**
  - 공통 코드 규칙 및 폴더 구조 수립 (파일/폴더 네이밍, import 순서, 코드 스타일 가이드)  
  - GitHub 브랜치 전략 및 커밋 컨벤션 정립
  - CSS 구조를 컴포넌트 단위로 설계하여 유지보수성 향상
- **웹 페이지 제작**
  - 로그인 / 회원가입 / 비밀번호 찾기 페이지 UI 제작  
  - 팀원 모집 페이지 레이아웃 및 반응형 구성  
  - Bootstrap을 활용한 폼 구성 및 기본 유효성 스타일링 구현
